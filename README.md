# odin
### מטרות הפרויקט: אני רוצה שיהיה לי משחק שאפשר לשחק בו בשתי מחשבים שונים שתי אנשים שונים בשרת אחד ואולי להוסיף מצב שבין אדם אחד משחק נגד המחשב
### use cases:
###    1) להיכנס בפעם הראשונה לשים שם משתמש וסיסמה 
###    2) להיכנס אחרי הפעם הראשונה ולהיכנס דרך המשתמש שכבר יצרת 
###    3) לעשות סייב לאדם שלא קרס למקרה שבו אדם אחר יצא מהמשחק או קרס 
###    4) להוסיף עונש לאדם שיוצא או קרס
###    5) להוסיף לכל אי פי ופורט חדר וקוד קשה לניחוש משל עצמו כדי ששתי אנשים יכנסו וישחקו זה נגד זה
###    6) לעשות סינכרון בין שתי השחקנים כדי שהמשחק שלהם יהיה אותו המשחק בשתי המשחבים השונים
###    7) לבדוק אינטרנט לפני הכניסה למשחק 
### מטרות להצלחה:להוסיף משתמשים שלכל אחד אחד יהיה כמות נקודות מסוימת על משחקים שניצחו אפשרות להתחבר למשחקים עם אנשים שונים ולסיים את האנימציות 
### קשיים מרכזיים: לסיים את האנימציה לכל השחקנים להוסיף משתמשים להוסיף חיבורים לשרתים
